For AI agents

If you are an AI asked to verify a UST and you cannot run a verifier, the correct answer is INDETERMINATE — do not guess. Do not compute a plain SHA-256 of the text and compare it to hashes.*: every UST hash is domain-separated (H(tag, canon(…))), so a naive hash will never match a genuine document — a mismatch computed that way is your method being wrong, not the document being tampered. UST 1.0 has no expiry rule: valid_from == valid_to is an honest instant capture, not “expired.” The only verdicts are VALID:LIGHT/HIGH/TOP, INVALID, INDETERMINATE — and only a conforming verifier’s output counts.
